# networking

## commands

### get IPs
```sh
# device IP - ipv4
curl -s https://api.ipify.org && echo

# device IP - ipv6
curl -s https://api6.ipify.org && echo
```

## apps

- **[proxyman](https://proxyman.com)** is a free app to capture, decrypt, and mock HTTP requests/responses with powerful debugging tools.

- **[nmap](https://nmap.org)** is a free and open-source network scanner. it is used to discover hosts, ports and services on a computer network by sending packets and analyzing the responses.
